# Tableau
## 주요 전처리 기능 정리


#### 1) 데이터 전처리
##### (1) 피벗 (long-form)
 : wide-form -> long-form데이터 형태여야 시각화 가능
```
Step_1) 전환해야하는 축 범위 선택
Step_2) 전환해야하는 축 마지막 컬럼에서 화살표 클릭
Step_3) 피벗 클릭
```


##### (2) 데이터 원본 필터링
```
Step_1) 데이터 원본 시트 우측 상단 '필터' '추가' 클릭
Step_2) 기준이 될 필드 선택
Step_3) 필터 기준 선택
```
